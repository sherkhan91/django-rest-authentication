Django Authentication

Welcome to django authentication using rest api.

This component is independent of backend.

There is top navigation bar on top where you can navigate to the desired functionality specific to your needs..

Home: tab is current page where you are and you will find all related information for navigation and usage.
Create: tab is where you can create new users, which you want to save into backend postgres user database.
Login: tab is where you can view login page and by providing your username and password you can login.
Update: tab is where you can update information of logged in user which include first name and last name.
Remove: tab is where you can remove the logged in user from database, you will no longer access to profile.
